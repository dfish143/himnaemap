# 힘내맵 메인 맵페이지
예랑가랑이 다녀간곳을 카카오 맵에서 확인하고, 해당장소에서의 촬영된 콘텐츠를 볼 수 있는<br> '힘내맵' 맵 페이지 (디자인 및 퍼블리싱)<br>
📌[홈페이지 바로가기](https://dfish143.github.io/himnaemap/index.html)<br>
<br>
<br>

## 기능 및 특징
- 힘내맨의 근원지, 부산대학교를 기준으로 펼쳐지는 '힘내맵'
- 장소정보를 클릭하여 영업시간, 연락처 등의 '장소정보(카카오맵,네이버지도)'를 열람
- 영상썸네일을 클릭하여 유튜브영상 등의 '촬영된 컨텐츠'를 열람
- 카테고리 필터링 버튼으로 원하는 카테고리만 표기가능
- 오른쪽 하단 '방명록' 바로가기 아이콘을 통해 의견 및 문의사항 공유가능
